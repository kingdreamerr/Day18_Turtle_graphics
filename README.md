## How to run this on your device

- Clone this repository
```
git clone https://github.com/kingdreamerr/Day18_Turtle_graphics.git
```
- cd into the repo
```
cd Day18_Turtle_graphics
```

- Paste the following in the terminal 
```
python3 main.py
```
